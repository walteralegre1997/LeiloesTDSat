# LeiloesTDSat

Descrição

Projeto desenvolvido para a atividade da disciplina (A1UC11), com o objetivo de aplicar o versionamento de código utilizando Git e GitHub.

O sistema é uma aplicação Java para gerenciamento de produtos de uma casa de leilões.

# Tecnologias utilizadas

- Java
- MySQL
- Git
- GitHub
- NetBeans IDE
