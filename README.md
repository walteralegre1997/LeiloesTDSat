\#LeiloesTDSat



\#Descrição



Projeto desenvolvido para a atividade da disciplina de versionamento utilizando Git e GitHub.



O sistema consiste em uma aplicação Java para gerenciamento de produtos de uma casa de leilões.



\#Tecnologias utilizadas



\- Java

\- MySQL

\- Git

\- GitHub

\- NetBeans
